# sentiment_analysis_lstm.ipynb
Sentiment analysis project using LSTM deep learning model

## Overview
This project focuses on sentiment classification using LSTM (Long Short-Term Memory) deep learning models.

The model analyzes text review data to classify sentiment categories and evaluate prediction performance.

## Tools
- Python
- TensorFlow / Keras
- Pandas
- NumPy
- Scikit-learn

## Process
1. Text preprocessing
2. Tokenization
3. Sequence padding
4. LSTM model training
5. Model evaluation
6. Sentiment prediction

## Result
The LSTM model successfully performed sentiment classification on review datasets with evaluation metrics such as accuracy and confusion matrix.
